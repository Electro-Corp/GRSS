# General Relativity Simulation Software (GRSS) 
Replace with a better name later. <br>
A simulation software for General Relativity. 

## Third-Party
* OpenGL
* OpenCL
* wxWidgets
* aixlogger (included, `GRSS/common/aixlog.h`)

## Source Tree
`GRSS/assets` contains visual assets <br>
`GRSS/common` contains files that are used by all components (eg. Vectors, Trigger/Connector manager) <br>
`GRSS/interface` contains the main entry point for the application, and handles all user interface tasks <br>
`GRSS/rendering` is a custom library for rendering <br>
`GRSS/physics` is a custom library for physics
