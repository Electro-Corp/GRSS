# General Relativity Simulation Software (GRSS) 
Replace with a better name later. <br>
A simulation software for General Relativity. 

## Source Tree
`source/assets` contains visual assets <br>
`source/common` contains files that are used by all components (eg. Vectors, Trigger/Connector manager) <br>
`source/interface` contains the main entry point for the application, and handles all user interface tasks <br>
`source/rendering` is a custom library for rendering <br>
`source/physics` is a custom library for physics
